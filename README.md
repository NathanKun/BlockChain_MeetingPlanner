# BlockChain_MeetingPlanner
ESIGELEC-IF-Promo2018-Blockchain project

## How to run
First run:
	npm install
  
First terminal:
	node_modules/.bin/truffle compile; node_modules/.bin/testrpc
  
Second terminal:
	node_modules/.bin/truffle migrate; node_modules/.bin/truffle serve
  
Navigator:
	http://localhost:8080/
